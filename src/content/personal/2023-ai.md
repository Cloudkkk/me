# 拥抱 AI 浪潮

将 AI 工具融入开发工作流，探索 LLM 的无限可能。

## 初次接触

2023 年，ChatGPT 的爆发让所有人重新审视 AI 的能力边界。作为开发者，我的第一反应不是恐惧，而是好奇——这东西到底能帮我做什么？

## 工作流变革

逐步将 AI 工具融入日常开发：

- **代码生成** — GitHub Copilot 成为日常编码的搭档
- **代码审查** — 用 LLM 做第一轮 Code Review
- **文档撰写** — 从技术文档到 API 说明，AI 大幅提升了效率
- **问题排查** — 复杂 Bug 的分析和定位

## 认知转变

> AI 不会取代程序员，但会用 AI 的程序员会取代不会用的。

这一年最重要的认知是：AI 是工具，不是答案。它放大了能力，但前提是你知道该问什么问题。
