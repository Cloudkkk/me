# 全栈开发实践

从前端到后端，TypeScript 全栈技术体系的构建与沉淀。

## 技术体系

这一年的核心目标是打通前后端的技术壁垒，构建一套统一的 TypeScript 全栈工作流：

- **前端** — React / Next.js / Tailwind CSS
- **后端** — Node.js / Express / tRPC
- **数据库** — PostgreSQL / Prisma ORM
- **部署** — Docker / Vercel / Cloudflare Workers

## 核心收获

全栈开发最大的价值不是 "什么都会"，而是 **端到端的思维方式**。当你同时理解前端的渲染机制和后端的数据流转时，架构决策会变得更加自然。

## 关键节点

- 第一个完整的全栈项目上线
- 建立了个人的 TypeScript 代码风格和项目模板
- 开始在团队中推动 Type-safe API 的实践
